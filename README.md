# yaseru

デブ脱出プロジェクト

https://github.com/tsu-nera/yaseru/blob/master/notebooks/weights_analysis.ipynb

## データ取得項目

* 体重
* 体脂肪率
* 筋肉量
* 内臓脂肪
* 基礎代謝量
* 体内年齢
* 推定骨量
* 消費カロリー
* 摂取カロリー
* 運動カロリー

## データ取得元

* [Fitbit](https://www.fitbit.com/)
* [HealthPlanet(タニタ)](https://www.healthplanet.jp/)
* [MyFitnessPal](https://www.myfitnesspal.com/ja/)
